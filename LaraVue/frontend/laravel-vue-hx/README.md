# LaraVue frontend template

This folder shows how the template repository explains the frontend stack.

## Stack

- Laravel for the server-side shell
- Vue for interactive panels
- HTMX for partial HTML swaps
- hyperscript for small event rules
- Tailwind CSS for styling

## Example usage

The sample page renders:

- a template repository notice
- a live visitor summary panel
- an HTMX partial refresh button
- a Vue component card
- a hyperscript-powered interaction

## Copy path

When you create a real Laravel project, copy the sample files into that project and adapt them to your needs.

```bash
composer create-project laravel/laravel sample
```
